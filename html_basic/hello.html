Manakala file itu dibuka oleh perangkat lunak code editor, Anda dapat melihat isi file dokumen html yang berisi perintah-perintah html. Perintah-perintah html dinamakan tag dan selalu ditulis dengan ciri diapit simbol sebagai berikut < >.

Ada dua jenis tag dalam html, yaitu tag tunggal dan tag berpasangan. Contoh tag tunggal misalnya <br>. Sedangkan contoh tag berpasangan misalnya <p> simbol tag di sebelah kiri adalah tanda tag pembuka, sementara di sebelah kanan paling ujung akhir dari baris kalimat ini adalah tag penutup. Di antara kedua tag pembuka dan penutup dapat dituliskan elemen web yang diatur oleh kedua tag tersebut </p>

Pada code editor biasanya terdapat keterangan nomer baris di sisi kiri laman editing dokumen. Nomor baris tersebut menjadi rujukan para penulis kode ketika mendiskusikan isi dokumen html atau file kode program lain

Sekarang mari kita bedah bagian-bagian yang tertulis dalam dokumen html namun tidak terlihat secara visual melalui browser.

Baris ke-1 adalah deklarasi bahwa file ini merupakan dokumen html versi terbaru. Anda dapat membaca informasi lebih dalam mengenai deklarasi ini di bagian komentar yang ditulis dalam dokumen html. Anda perlu membuka file ini dengan code editor terlebih dahulu

Baris ke-2 <html> adalah tag penanda awal mulainya dokumen html seperti yang dinyatakan pada baris pertama. Dokumen html tersebut berakhir sampai pada baris yang ditandai dengan tag </html>

Di dalam tag <html> </html> terdapat dua bagian utama, yaitu head dan body. 

    <head>
    Isi bagian yang diapit tag head biasanya diisi dengan meta data dokumen web. Meta data adalah informasi yang hanya diperuntukkan bagi browser mengenasi aspek teknis dan spesifikasi dokumen html yang terkait. Bagian ini tidak akan ditampilkan secara visual oleh browser.

    Bagian head yag biasanya tidak terlewatkan untuk diisi adalah:
    <title> Judul laman web yang akan ditampilkan di tab browser</title>
    </head>

    <body>
    Elemen-elemen web yang dituliskan diantara tag pemuka dan penutup body akan dibaca dan dirender oleh browser menjadi tampilan visual atau bahkan audio.
    </body>
